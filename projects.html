Credit Card Fraud Detection

2026

Engineered a dual-pipeline system to handle class imbalance in high-dimensional streaming data. This project benchmarks Scikit-learn against RiverML for real-time anomaly detection using Isolation Forest algorithms.

The system includes a Streamlit prototype for visualizing detection patterns and monitoring financial loss mitigation in real-time. Key focus was on handling the trade-off between false positives and false negatives in production environments.

Implemented real-time signal processing pipeline to interpret electromyography (EMG) sensor data for precise motor actuation. This allows users to control the prosthetic through natural muscle signals, providing a more seamless and responsive interaction experience.
